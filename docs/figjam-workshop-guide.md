## Tipps für FigJam

- **Sections**: Rechtsklick → Add Section, dann Farbe anpassen
- **Stickies**: `S` drücken oder Insert → Sticky Note
- **Sticky-Farben**: Klick auf Sticky → Farbpalette oben
- **Text fett**: Sticky-Text markieren → Cmd+B
- **Pfeil zwischen Sections**: `C` drücken (Connector)
- **Emoji**: direkt in Text tippen oder Windows+. / Cmd+Ctrl+Space

**FigJam-Farb-Codes für Stickies:**
- 🔴 Rot = Hochpriorität
- 🟠 Orange = Mittlere Priorität
- 🟡 Gelb = Stärken / Infos
- 🟢 Grün = Positiv / Phase 2
- 🔵 Blau = Phase 1 / Keywords
- 🟣 Lila = Zielgruppen
